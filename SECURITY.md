# 安全策略

LinkGallery 会读取用户的私人照片和视频，因此安全问题优先于功能便利。

请不要在公开 Issue 中披露可导致未授权媒体访问、配对绕过、密钥泄露或路径穿越的
细节。请通过 GitHub Security Advisory 私下报告，并包含复现步骤、影响版本和建议
修复方向。

## 安全基线

- 未配对设备不能读取媒体元数据、缩略图或原文件。
- Android API 不提供上传、删除、移动、重命名或编辑媒体的能力。
- 配对密钥必须存入系统安全存储，不得出现在日志或崩溃报告中。
- 媒体 ID 由服务端解析，客户端提供的路径不能直接进入文件系统 API。
- 下载先写入 `.partial`，校验成功后再原子重命名。
- 日志默认隐藏令牌、证书、公钥指纹以外的密钥材料和完整媒体路径。
