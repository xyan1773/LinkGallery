# 产品范围

## 一句话定位

在同一局域网内，从 Windows 安全、只读地浏览 Android 相册，并可靠复制原文件。

## MVP 用户旅程

1. 用户在 Windows 和 Android 安装 LinkGallery。
2. 两端通过二维码完成一次配对。
3. 之后处于同一 Wi-Fi 时自动发现并连接。
4. Windows 按时间线浏览手机照片和视频。
5. 用户选择文件复制到电脑，可暂停、恢复和断点续传。
6. 手机离线后，Windows 仍可搜索已缓存的媒体索引和缩略图。

## MVP 包含

- Android / Windows 配对和局域网发现；
- MediaStore 分页读取；
- 缩略图缓存、照片预览、基础视频播放；
- 单个和批量复制、断点续传、完成校验；
- 去重、离线状态、Pocket 3 来源识别。

## MVP 不包含

iPhone、公网访问、云同步、媒体编辑、人脸识别、AI 搜索、电脑向手机上传、手机
媒体整理，以及 Windows 直接连接 Pocket 3。

## 不可妥协约束

- 浏览与复制权限分开理解，手机端只输出读取流。
- 不在 Android 端实现任何媒体写操作。
- 不以文件名作为媒体身份或去重的唯一依据。
- 传输失败不能留下看似完整的损坏文件。
